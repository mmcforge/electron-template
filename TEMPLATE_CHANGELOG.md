# Template Changelog

All notable changes to the MMC Forge Electron Template will be documented in this file.

## [1.0.0] - 2024-11-08

### Added
- Initial MMC Forge template based on Guasam's electron-react-app v12.0.0
- Interactive setup script (`setup-template.js`) for easy customization
- Comprehensive template usage documentation
- GitHub template repository configuration
- Updated branding and attribution

### Changed
- Package name from "era" to "mmcforge-electron-template"
- All references updated to MMC Forge branding
- Repository URLs updated to MMC Forge organization

### Removed
- Template-specific lock files (users generate their own)
- Original git history (clean start for new projects)

### Features Inherited from Original
- Electron 37.3.1 with React 19
- Vite for fast development and building
- TypeScript for type safety
- Shadcn UI component library
- TailwindCSS for styling
- Conveyor IPC system for type-safe communication
- Custom window titlebar and menus
- Dark/light theme switching
- Cross-platform build support
- VS Code debugging configuration

**Original Template**: Based on [electron-react-app](https://github.com/guasam/electron-react-app) by [Guasam](https://github.com/guasam)
